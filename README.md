TS Academy Assignment on Building a Fntech (EBI Bank) with External  API

Tech Stack:
- Node.js
- Express.js
- MongoDB (with Mongoose)

  Aim: This assignment is centered on improving ideas on working with External API. With that, I built a Fintech platform
  (EBI Bank) that operates like a regular bank. Before a customer creates an account, he/she will need a valid identity (BVN/NIN)
  wich contains certain basic information.

  ! (EBI Bank) then onboarded the customer, he/she moves on to create an account to get account number, account number, etc. With the
  account number, the customer can verify his/her account name, and upon successful creation of account, each customer is credited 15,000.

  The project supports transfer of funds from one account to the other, balance check, transaction reference verification.

  - For the safety of customers password during onboarding, I utilized bcrypt for hashing of password. With this, customers are assured
  - that their data are private.

- Also, I made use of MongoDB to store customers details for full accountability of customers information.

- For API documentation, I made use of Postman to run all the endpoints and to generate a token for each transaction to go through.
